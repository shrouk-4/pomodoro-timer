# Pomodoro Timer ⏰

This is a stylish and customizable Pomodoro Timer built with HTML, CSS, and JavaScript.

## ✨ Features

- 25-minute work session
- 5-minute break timer
- Ambient background sounds (Ocean, Rain, Forest, etc.)
- Background themes with automatic transitions
- Responsive design with side panel options
- Beautiful fonts and smooth UI
